# Project-Specific Guidelines

## Project Overview
**Name:** slack-notifications
**Description:** A simple Python library for Slack notifications at application milestones.
**Tech Stack:** Python 3.8+, Slack SDK, Pydantic.

## Development Guidelines

### Structure
- `src/`: Source code.
- `tests/`: Pytest tests.
- `dev_notes/`: Project plans and change logs.
- `docs/`: Documentation.

### Naming Conventions
- Follow Python PEP 8.
- Use `snake_case` for variables and functions.
- Use `CamelCase` for classes.

### Prohibited Actions
- Do not modify `docs/system-prompts/`.
- Do not hardcode API tokens.

### Getting Help
- If you encounter ambiguous requirements or technical blockers, **STOP** and ask for clarification.
